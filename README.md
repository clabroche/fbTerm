# FbTerm

![Demo](./demo.gif)
## Install
``` bash
    git clone git@github.com:clabroche/fbTerm.git
    cd fbTerm
    npm i 
```

## Usage
``` bash
    npm run start 
```

On first start cli ask to create credentials
Credentials are store and encrypted in credentials.json
